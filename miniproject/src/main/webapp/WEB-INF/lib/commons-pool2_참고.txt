* JNDI 방식의 DBCP 사용

	- commons-dbcp2-2.11.0.jar
	- commons-pool2-2.12.0.jar
	- commons-loggin-1.3.jar
		-> org.apache.commons.dbcp2.BasicDataSource
			클래스에서 logging 라이브러리에 있는 클래스를 사용하기 때문에 필요하다.
			이렇게 어떤 클래스에서 다른 클래스를 사용하는 것을 그 클래스에 의존하고 있다고 말한다.
			
			commons-loggin-1.3.jar 파일이 없으면 아래와 같은 Exception이 발생한다.
			javax.servlet.ServletException: javax.naming.NamingException: Could not create resource factory instance [Root exception is java.lang.NoClassDefFoundError: Could not initialize class org.apache.commons.dbcp2.BasicDataSourceFactory]
			
			
	- context.xml에 factory 설정은 아래와 같이 한다. 
	  factory="org.apache.commons.dbcp2.BasicDataSourceFactory"
